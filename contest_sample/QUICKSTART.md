## 🛠️ If Something Goes Wrong

| Error | Fix |
|-------|-----|
| `ModuleNotFoundError` | Run `pip install -r requirements.txt` |
| Port 5000 in use | Change port in app.py (line ~377) |
| Can't import app | Run from contest_sample folder |
| Charts not showing | Refresh browser (Ctrl+Shift+R) |
| CSV upload fails | Use format: name,category,price,rating |
